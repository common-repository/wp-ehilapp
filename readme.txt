Il plugin serve per  inserire il proprio sito nell'app per smartphone Ehilapp

== Description ==
Se si vuole inserire il proprio sito nella lista dell'app per smartphone Ehilapp bisogna configurare questo plugin e soprattutto inserire il codice personale dell'utente ottenuto tramite app.

== Installation ==
Fai l'upload del plugin e attivalo. Poi inserisci il codice personale che trovi nell'app, seleziona le categorie che vuoi mostrare ai tuoi utenti ed Ã¨ fatta.
